README.txt

The following directories and files are contained in the tarball LDAS_prep.tgz:

LDAS_SOIL_VIC.TAR.gz

	LDAS soil parameter file and VIC soil file preparation programs

LDAS_VEG_VIC.TAR.gz

	LDAS Vegetation parameter file and VIC vegetation parameter file
preparation programs

LDAS_SNOWBAND_VIC.TAR.gz

	LDAS snowband preparation programs (currently set for 6 snowbands in
output file)

extract_daily_means_from_LDAS_binary.c

	reads LDAS binary and extracts daily average or summed values for all
model output variables for the specified period

quick_read_LDAS_format.c

	quick read of binary LDAS flux file format, output sent to stdout as
ASCII columns

LDAS_2_vegparam.c

	creates a VIC vegetation parameter file without root zones from the
ASCII file of UMD LDAS vegetation cover categories
