The following files are contained in the tarball spatial_disagg_precip.tgz:

convert_srf_to_text.scr

	converts an .srf to text

make_arc30_II_data.scr

	disaggregates *.scr file from 5 minutes to 30 arc seconds

run_soil_aggr.scr

	aggregates 30 arc seconds to 1/8 degree

Scalesrf.c

	aggregates 30 arc seconds to 1/8 degree

make_my_II_data.scr

	creates for each variable and depth a file with the lat, long, and the
corresponding variable value
